🚢 Ship Performance Dashboard (Power BI)
📌 Overview

This project is a Power BI dashboard designed to analyze and monitor ship performance across different vessel types such as:

Container Ships
Tankers
Bulk Carriers
Fish Carriers

It provides insights into operational efficiency, fuel usage, speed, and maintenance status.

📊 Features
📈 Interactive dashboards with filters and slicers
🚢 Separate pages for each ship type
⚡ Performance metrics (Speed, Distance, Engine Power)
🔧 Maintenance status tracking
🗺️ Route type analysis (Sea / Port / Transit)
🛠️ Tools & Technologies
Power BI – Data visualization
Power Query (M Code) – Data transformation
Excel / CSV – Data source
📂 Dataset

The dataset includes:

Ship Type
Route Type
Engine Type
Maintenance Status
Speed Over Ground (knots)
Engine Power (kW)
